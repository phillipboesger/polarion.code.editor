avaCopilot - AI Agent

AI-based assistance by avasis Solutions GmbH

What is avaCopilot?

avaCopilot is the AI agent of avasis Solutions GmbH. It is designed to integrate AI features into Polarion by providing best-practice features from avasis as well as customer-specific extensions.

avaCopilot receives data from the Polarion API or UI, combines it with a defined prompt and sends it to the configured Large Language Model (LLM). The results are processed and presented to the user in a structured way.

Privacy & Security

Handling of data:

  • Data is not stored by avasis Solutions GmbH
  • Data is forwarded to the configured LLM provider
  • Privacy and security settings must be configured at the LLM provider
  • Sensitive information should always be handled with care

Note: For questions about data protection, configuration or usage of avaCopilot, please contact avasis Solutions GmbH. You are welcome to send us your feedback.
